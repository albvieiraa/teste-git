# Day 47 - Top Trumps

## Objetivo 🎯
- Criar um jogo no estilo "Super Trunfo".

## O que fiz 💻

- Um dicionário de cartas
- Variável com a pontuação dos jogadores
- Loop `while True` para o jogo rodar indefinidamente até que os jogadores decidam parar.

## Como executar ⏏️▶️
No terminal, execute:
```bash
python main.py
```
